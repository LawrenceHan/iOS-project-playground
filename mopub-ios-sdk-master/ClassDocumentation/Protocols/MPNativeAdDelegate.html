MPNativeAdDelegate Protocol Reference

Conforms to NSObject
Declared in MPNativeAdDelegate.h

Overview

The delegate of an MPNativeAd object must adopt the MPNativeAdDelegate protocol. It must implement viewControllerForPresentingModalView to provide a root view controller from which the ad view should present modal content.
