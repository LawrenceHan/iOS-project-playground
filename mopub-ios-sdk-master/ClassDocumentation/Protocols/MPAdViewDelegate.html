MPAdViewDelegate Protocol Reference

Conforms to NSObject
Declared in MPAdView.h

Overview

The delegate of an MPAdView object must adopt the MPAdViewDelegate protocol. It must implement viewControllerForPresentingModalView to provide a root view controller from which the ad view should present modal content.

Optional methods of this protocol allow the delegate to be notified of banner success or failure, as well as other lifecycle events.
